# 鸽子要跑路

# 题目描述

鸽子位于平面直角坐标系上的整点 (a,b)(a, b)(a,b)，每次可以朝上下左右四个方向选一个走一个单位长度，走的时候需要保证横纵坐标非负，目的地是 (c,d)(c, d)(c,d)。
鸽子的初始疲劳值为 000，如果当前位置 (x,y)(x, y)(x,y) 满足 x&y > 0 ，那么疲劳值加 111，否则不变。起点与终点的疲劳值不计入答案。
一共 T(1≤T≤105)T(1 \le T \le 10^5)T(1≤T≤105) 组数据，对于每组数据求从 (a,b)(a, b)(a,b) 到 (c,d)(c,d)(c,d) 的最小疲劳值。(0≤a,b,c,d≤1018)(0 \le a,b,c,d \le 10^{18})(0≤a,b,c,d≤1018)。

# 解题思路

是一道找规律题，先打一手表：

发现是个 谢尔宾斯基三角形。
答案有两种情况，一种是直接走，每一步都算入疲劳值；另一种是先走到一个 000 上，再找个最近的位置走到目标点 (所有的 000 都是联通的)
对于一个正方形的左上部分，是一种递归结构，可递归求解。当落入某个正方形的右下时，计算与边界的距离。

# abab

# 题目描述

给一个只包含 a 和 b 的字符串，问最多有多少不相交的子序列 abab 。
共 TTT 组数据，每组数据包含一个字符串 sss。(1≤T≤5,1≤∣s∣≤107)(1 \le T \le 5, 1 \le |s| \le 10^7)(1≤T≤5,1≤∣s∣≤107)

# 解题思路

abab 可以拆成两个 ab , 所以问题就转化为了找出尽量多的不相交的 ab 。
记字符串中， ab 相互交错的数量最多的为 mxmxmx，一共能找到 cntcntcnt 对 ab 。

  1. 当 mx>⌊cnt2⌋mx > \left\lfloor\frac{cnt}{2}\right\rfloormx>⌊2cnt​⌋ 时，会剩下相互交错的不能匹配的 ab ，答案为 cnt−mxcnt-mxcnt−mx；
  2. 当 mx≤⌊cnt2⌋mx \le \left\lfloor\frac{cnt}{2}\right\rfloormx≤⌊2cnt​⌋ 时，所有的 ab 一定能找到一种方案全部匹配，答案为 ⌊cnt2⌋\left\lfloor\frac{cnt}{2}\right\rfloor⌊2cnt​⌋

# 二叉树比大小

# 题目描述

用 000 代表空树，如果一个节点没有左子树，那么就认为它的左子树是空树（右子树同理）。对于任意二叉树 AAA，0≤A0 \le A0≤A 成立；对于任意非空二叉树 AAA，A≤0A \le 0A≤0 不成立。
用 lslsls 代表左子树，rsrsrs 代表右子树，对于所有非空二叉树 AAA，BBB，A≤BA \le BA≤B 当且仅当 ls(A)≤ls(B)ls(A) \le ls(B)ls(A)≤ls(B) 且 rs(B)≤rs(A)rs(B) \le rs(A)rs(B)≤rs(A)；A=BA = BA=B 当且仅当 A≤BA \le BA≤B 且 B≤AB \le AB≤A，否则 A≠BA \not = BA=B。
给定节点编号为 1∼n1 \sim n1∼n 的二叉树 AAA。 BBB 的初始形态为 AAA。对 BBB 重复进行以下操作：任选 BBB 中的一个儿子数量不超过 111 的节点，为这个节点增加一个儿子（使得 BBB 仍然是二叉树）。左右有区别。
问有多少不同的 BBB 满足 BBB 的节点数为 n+mn+mn+m 且 A≤BA \le BA≤B。

# 解题思路

如果 A≤BA \le BA≤B ，那么 AAA 的每一个子树 A′A'A′ 和它对应的 B′B'B′，都有大小关系的限制。这个限制要么是 A′≤B′A' \le B'A′≤B′，要么是 A′≥B′A' \ge B'A′≥B′。我们可以在要求 A′≤B′A' \le B'A′≤B′ 的子树的根节点上标记状态 111 ，在要求 A′≥B′A' \ge B'A′≥B′ 的子树的根节点上标记状态 000 。左儿子的状态和父节点相同，右儿子的状态和父节点相反。
我可以且只可以在状态为 111 的地方添加节点。可以用一次 dfs 找到所有可以添加新节点的位置。我们只关心这些位置的数量，设为 ccc。
问题转化成这样：

有 个互相独立的位置，放上 个节点，问最终形成的二叉树森林有几种。

记 dp[c][m]dp[c][m]dp[c][m] 为上述问题的答案。

分类讨论第 ccc 个位置上是否挂了新的节点，如果挂了一个节点，那么还需要的节点数少了 111，一共需要挂的位置多了 111；如果不挂这个节点，后面就不考虑他了，一共要挂的位置就少了 111。
所以转移方程为 dp[c][m]=dp[c+1][m−1]+dp[c−1][m]dp[c][m] = dp[c+1][m-1]+dp[c-1][m]dp[c][m]=dp[c+1][m−1]+dp[c−1][m]。
把他看成在二维平面上的行走问题，则每次可以从 (x,y)(x,y)(x,y) 走到 (x+1,y−1)(x+1,y-1)(x+1,y−1) 和 (x−1,y)(x-1,y)(x−1,y) (需要保证 x≥0x \ge 0x≥0)，问题是从 (1,0)(1,0)(1,0) 走到 (c+m,m)(c+m,m)(c+m,m)
然后把坐标转化一下，将 (x,y)(x,y)(x,y) 转化为 (x+y,y)(x+y,y)(x+y,y)， 那么从 (x,y)(x,y)(x,y) 往后走一步走到的点就是 (x+1,y)(x+1,y)(x+1,y) 和 (x,y+1)(x,y+1)(x,y+1)，同时上面的限制就变成了要保证 x≤yx \le yx≤y，而问题是从 (0,1)(0,1)(0,1) 走到 (m,c+m)(m,c+m)(m,c+m)。总方案是 (c+2m+1m)\binom{c+2m+1}{m}(mc+2m+1​)，枚举每第一次超出限制的点，就和 No.3 Count 那道题一样，得到最后的答案为 (c+2m−1m)−(c+2m−1c+m)\binom{c+2m-1}{m}-\binom{c+2m-1}{c+m}(mc+2m−1​)−(c+mc+2m−1​).
