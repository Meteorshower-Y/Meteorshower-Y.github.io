# 最短子列

# 题目描述

对于两个字符串 SSS， TTT 我们定义它们的最短公共非子序列为一个最短的字符串，使得它既是 SSS 的非子序列，也是 TTT 的非子序列。

这里 SSS,TTT 仅由 000 和 111 构成。现在请你求出字典序最小的最短公共非子序列。

# 解题思路

一定有的边是原树边，一条非树边要么不能加，要么加不加都可以，而且能加的非树边互不影响，因为一定不会遍历到。所以答案是 2cnt2^{cnt}2cnt，cntcntcnt 为可以加的非树边的数量。
一条非树边 (u,fa(v))(u,fa(v))(u,fa(v)) 可以连的条件为 vvv 为 uuu 的祖先，vvv 的权值 小于 uuu。所以直接在 dfs 的时候用树状数组统计路径上权值的前缀和即可。
