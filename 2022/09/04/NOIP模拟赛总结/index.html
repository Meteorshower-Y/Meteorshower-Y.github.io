# One to one

# 题意描述

给定一个数组 XiX_iXi​ ，F (X) 值的定义为，在一个第 iii 条边为 (i,Xi)(i, X_i)(i,Xi​) 的无向连通图的连通块个数。
现有一个序列 aia_iai​ ，一些位置为 −1-1−1 ， 则这个位置可以为 111 ~ nnn 中的任意一个数，问所有可能的 aaa 的 F(a)F(a)F(a) 之和对 998244353998244353998244353 取模的值。
n≤2000n \le 2000n≤2000

# 解题思路

对于给出的边，可以构成若干个 树和环 (奇环树)。
如果是一棵奇环树，那么这个连通块内所有的边都已经确定，可以直接算贡献。记奇环树个数为 aaa，树的数量 (-1 的个数) 为 bbb， 则贡献为 a×nba \times n^ba×nb。
如果是一棵树，则仅有一条边为确定 (根的边)。记 fi,jf_{i,j}fi,j​ 表示前 iii 棵树，选 jjj 棵组成一个连通块的方案数，有转移方程如下： fi,j=fi−1,j+fi−1,j−1×sizeif_{i,j} = f_{i-1,j} + f{i-1, j-1} \times size_ifi,j​=fi−1,j​+fi−1,j−1×sizei​。若共有 kkk 棵树，答案为 ∑i=1kfk,i×nk−i(i−1)!\sum\limits_{i=1}^k f_{k,i} \times n^{k-i}(i-1)!i=1∑k​fk,i​×nk−i(i−1)!。

# D&D

# 题意描述

给定不可重集合 AAA，定义其伴随子集为：
f(A)={a∈a∣∀b∈A−{a},a∣b≠b}f(A) = \{a \in a | \forall b \in A-\{a\}, a | b \not= b \}f(A)={a∈a∣∀b∈A−{a},a∣b=b}

给定一个长度为 nnn 的序列 aia_iai​，把这个序列分成连续的若干个子段，要求这些子段的不重集的伴随子集相同。
输出方案数对 109+710^9+7109+7 取模的结果。

# 解题思路

记原序列的伴随子集为 FFF， 这有几个结论：

  1. 若 x∈Fx \in Fx∈F, 则 xxx 在子段的伴随子集中也出现。

  2. 若 x∉Fx \not\in Fx∈F, 则 xxx 在子段的伴随子集中也不会出现。

  3. 划分完子段的伴随子集与原序列的伴随子集相同。

  4. 若 BBB 为原序列的子集，且 F⊆BF \subseteq BF⊆B, 则 FFF 为 BBB 的伴随子集。

第一步我们需要快速求出原序列的伴随子集。
我们对 aaa 从大到小排序，依次 dfs 标记处二进制下被 aia_iai​ 包含的数。
如果 aia_iai​ 在 dfs 开始时没有被标记，他就是伴随子集中的一员。

第二步考虑计算方案。
若区间 [l,r][l,r][l,r] 有合法的伴随子集，那么区间 [l−1,r][l-1,r][l−1,r] 也有。我们就需要求出对于每一个 rrr, 满足条件的最大左端点 LiL_iLi​ ，按照 fi=∑i=0Li−1fjf_i = \sum\limits_{i=0}^{L_i-1}f_jfi​=i=0∑Li​−1​fj​ (前缀和优化)。

# Count

# 题意描述

现有 nnn 个红珠子和 An+BAn+BAn+B 个蓝珠子，从左往右放置，要保证在任何时候，红柱子有 xxx ，蓝珠子不超过 Ax+BAx+BAx+B 个，请你求出摆完所有珠子的方案数。
答案对 109+710^9+7109+7 取模。

# 解题思路

题意可以转化为在一个 n×mn \times mn×m 的网格图中，从 (0,0)(0,0)(0,0) (左下角) 走到 (n,m)(n,m)(n,m) (右上角)，不穿过直线 y=Ax+By = Ax+By=Ax+B 的方案数。

若不考虑直线的限制，方案总数为 Cn+mnC_{n+m}^{n}Cn+mn​。

考虑不合法的方案数：
记第一次走到直线的位置为 (p,Ap+B)(p, Ap+B)(p,Ap+B)，然后穿过直线，走到 (p,Ap+B+1)(p, Ap+B+1)(p,Ap+B+1)，然后再走到终点的方案数为

Cm−Ap−B−1+n−pn−p=(m−Ap−B−1+n−p)!(m−Ap−B−1)!(n−p)!=(m−Ap−B−1+n−p)!(m−Ap−B)!(n−p−1)!×m−Ap−Bn−p=A×Cm−Ap−b−1+n−pn−p−1\begin{aligned} C_{m-Ap-B-1+n-p}^{n-p} &= \frac{(m-Ap-B-1+n-p)!}{(m-Ap-B-1)!(n-p)!} \\ &= \frac{(m-Ap-B-1+n-p)!}{(m-Ap-B)!(n-p-1)!} \times \frac{m-Ap-B}{n-p}\\ &= A \times C_{m-Ap-b-1+n-p}^{n-p-1} \end{aligned} Cm−Ap−B−1+n−pn−p​​=(m−Ap−B−1)!(n−p)!(m−Ap−B−1+n−p)!​=(m−Ap−B)!(n−p−1)!(m−Ap−B−1+n−p)!​×n−pm−Ap−B​=A×Cm−Ap−b−1+n−pn−p−1​​

化简后的式子结果等于从 (p,Ap+b+1)(p, Ap+b+1)(p,Ap+b+1) 走到 (n−1,m+1)(n-1,m+1)(n−1,m+1) 的方案数。

记从 (0,0)(0,0)(0,0) 走到 (p,Ap+B+1)(p, Ap+B+1)(p,Ap+B+1) 的合法方案数为 WpW_pWp​, 不合法的方案总数为 ∑p=0n−1Wp×Cm−Ap−b−1+n−pn−p−1=Cn+mn−1\sum\limits_{p=0}^{n-1}W_p \times C_{m-Ap-b-1+n-p}^{n-p-1} = C_{n+m}^{n-1}p=0∑n−1​Wp​×Cm−Ap−b−1+n−pn−p−1​=Cn+mn−1​ (相当于从原点走到 (n−1,m+1)(n-1,m+1)(n−1,m+1) 的方案数)。

# Expand

# 题目描述

刺豚小 T 出去买菜，在 n×mn \times mn×m 的坐标系上，现在在位置 (x,y)(x,y)(x,y), 坐标系上有一些菜店和障碍，他需要在所有菜店买菜。他比较喜欢大的体格 (膨胀)，定义体格为 xxx 的他占据的格子数为 2x+1×2x+12x+1 \times 2x+12x+1×2x+1。他最大体格为 sss (膨胀的上限)。在不考虑回去的情况下，找一条路径，在满足路径最短的前提下，走到每个格子的体格和最大。
请输出最短的距离和满足最短距离下最大的体格和。
n,m≤300;  s≤10;  p≤15n,m \le 300; \; s \le 10; \; p \le 15n,m≤300;s≤10;p≤15

# 解题思路

首先可以预处理出每个格子能做到的最大体格，可以枚举判断，也可以从每个障碍出发进行 bfs。
然后再图上跑多源最短路 (其实就是 p+1p+1p+1 遍 dijkstra)，以距离为第一关键字 (取较小值为优)，路径体格和为第二关键字 (取较大值为优)，把 ppp 个菜市场和起点之间两两最优路径求出来。
要求每个菜市场都到过，显然重复经过仅去一个去过的不优，ppp 的范围很小，状压就好了。

# Birthday

# 题目描述

有 nnn 个小朋友，每个小朋友的礼物的初始价格为 aia_iai​，价格的上限为 vvv。
有两种操作共 mmm 个：

  1. 操作一：在区间 [l,r][l,r][l,r]，选择两个不相交的子集 (但不一定全都选上)，使两个集合对应的小朋友的礼物花费和相等 (第 iii 个小朋友礼物的花费为 ai+1a_i+1ai​+1)。如果存在这么一个集合就输出 Yes ，否则输出 No 。
  2. 操作二：使区间 [l,r][l,r][l,r] 的每个小朋友礼物价值变为 ax3%va_x^3 \% vax3​%v。

n,m≤105;  v≤1000;  a[i]<vn,m \le 10^5; \; v \le 1000; \; a[i] < vn,m≤105;v≤1000;a[i]<v

# 解题思路

考虑操作一：
当区间足够大的话是一定有合法方案的。
若区间长度为 kkk，那么子集就有 2k2^{k}2k 种选择方法，一种选法价格和的值域在 [1,v∗k][1,v*k][1,v∗k] 之间。考虑抽屉原理， 2k>v∗k2^{k} > v*k2k>v∗k 时，就至少有一种值的方案最少为两种（就算相交，减去相交部分也相等），这时候就一定有合法方案。解出 k≥14k \ge 14k≥14。
当 k≤13k \le 13k≤13 时，我们只能暴力统计，但 3k3^k3k 的复杂度难以接受，我们可以将区间砍成两半（折半搜索），将左半部分的可能取值存起来，再在搜右半部分是判断有没有取值相等的选法。当然，搜到了和为 000 的选法也可以直接 return。
单词询问的时间复杂度最差为 O(37+36)O(3^7 + 3^6)O(37+36)。

考虑操作二：
将乘法变为加法，即每个数进行的操作二的次数，值域为 [0,1000)[0,1000)[0,1000)，可以倍增预处理求值，用线段树维护区间加，每次将用到的数的标记下传，毕竟一次最多用 131313 个数。
